Aperçu
Tic Tac Toe with AI est une application permettant de jouer au Morpion contre une IA qui utilise l'algorithme Minimax pour jouer de manière optimale. L'IA peut être configurée pour jouer en tant que X ou O, et le jeu gère les égalités et les victoires.

Fonctionnalités
Jeu en un joueur contre l'IA
IA basée sur Minimax (jeu optimal)
Sélection du joueur humain (X ou O)
Réinitialisation de la partie
Affichage du résultat (victoire, défaite, égalité)
Interface simple et responsive

Installation
Clone le dépôt :
git clone https://github.com/TON_UTILISATEUR/Tic-Tac-Toe-with-AI.git
Place-toi dans le dossier du projet :
cd Tic-Tac-Toe-with-AI
Installe les dépendances (si un package.json existe) :
npm install
Lance le projet en local (selon la configuration) :
npm start
ou ouvre index.html dans ton navigateur.
Remplace les commandes selon la configuration réelle du projet (parcel, webpack, vite, etc.).
